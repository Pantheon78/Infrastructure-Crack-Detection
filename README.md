# Infrastructure-Crack-Detection

A computer vision-based infrastructure inspection system for detecting structural cracks from images and videos using OpenCV and Python.

## Features
- Crack detection
- Contour analysis
- Severity estimation
- Video processing
- Automated inspection support

## Technologies
- Python
- OpenCV
- NumPy

## Future Improvements
- YOLOv8 integration
- AI classification
- Automated reports
